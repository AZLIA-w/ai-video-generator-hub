# Now_trending - 热词快站项目

基于热词快站教程的项目模板。

## 快速开始

### 1. 选择热词
使用以下方法发现热词：
- Google Trends (trends.google.com)
- Twitter/X 关注 AI 领域的博主
- Hugging Face Spaces
- B站 AI 新闻 UP 主

### 2. 替换占位符
编辑 `src/index.html` 中的所有 `[PLACEHOLDER]`，参考 `config/placeholders.json`

### 3. 内容要求
- [x] 关键词密度：2-5%
- [x] 字数：至少 1200 词
- [x] 必须包含：What is / Why / How to / FAQ

### 4. 部署
```bash
# 推送到 GitHub
# 在 Vercel 导入仓库
# 绑定自定义域名
# 提交到 Google Search Console
```
